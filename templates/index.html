About This Tool

Educational tool that captures live network traffic using Scapy, analyzes packet behavior, and generates a fingerprint to help students compare how websites communicate.

🐍 Python📡 Scapy🧪 Flask 📊 Chart.js⚡ JavaScript🎨 HTML/CSS

Network Analysis Workflow

  1. User enters URL(s) and clicks Analyze
  2. Flask backend resolves domain to IPs
  3. Scapy captures packets for N seconds
  4. extract.py computes all metrics
  5. fingerprint.py builds structured JSON
  6. classify.py assigns behavior label
  7. JSON returned to frontend
  8. Charts & summary rendered in browser

About

This educational tool captures live network traffic with Scapy, extracts features, builds a structured network fingerprint, and classifies website behavior into categories like Streaming, Social Media, Static Content, or API-Heavy.

Tech stack: Python • Scapy • Flask • HTML/CSS • JavaScript • Chart.js • pcap/JSON.
